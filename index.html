React/Redux dev tools

Для чого нам потрібні

  • Debugging
  • Refactoring
  • Audit

Components

  • Дозволяє побачити дерево компонентів
  • Дозволяє побачити властивості та хуки для обраного компонента
  • Виділяє обраний компонент на сторінці
  • Знаходить компонент при кліку на нього на сторінці

Profiler

  • Профілювати та аналізувати зміни в компонентах з старту додатку
  • Профілювати та аналізувати зміни в певних проміжках
  • Відображає кількість рендерів певного компонента та причини
  • Відображає час, який витрачений на рендеринг компонента
